# LINALABS Console

Sitio web interactivo de LINALABS diseñado como consola alienígena. Experiencia inmersiva con navegación basada en RGB, efectos 3D isométricos en scroll, y tracking del mouse.

## Features

- 🎮 Consola alienígena interactiva con logo central responsivo al mouse
- 🌈 Sistema de colores RGB puro para 4 servicios
- 📜 Scroll vertical con reveales progresivos y animaciones suaves
- 📱 Responsive design (mobile, tablet, desktop)
- ♿ Accesibilidad WCAG 2.1 AA
- ⚡ Performance optimizado (LCP < 1s)

## Servicios

1. **Diseño Gráfico & Dirección Creativa** (Rojo #FF0000)
2. **Producción Audiovisual** (Verde #00FF00)
3. **Coaching Estratégico** (Azul #0000FF)
4. **Interim Management** (Multicolor RGB)
